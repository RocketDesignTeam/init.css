init.css

The perfect style for your project or reading pages

View project onGitHub

A css stylesheet that needs less HTML structure, for better result .

Introduction

With init.css, you only focus on inline tags. No need to add divs, or any bloc elements. If you want a kind of gihub pages site for a project and you only want to write the text.

Documentation

The header

The header will take the main color of the theme, its generated by the pseudo classes
:before / :after applied to the body, to create a huge header and introduce your document Nicely

The Title

Its really simple, the title of your document will be the first H1 of the page, and it'll stand on the header.

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
        <h1> Title of your doc </h1>
    </body>
</html>

Header image

If you want to add a cool header image to give more power to your page, there is an easy way to do it.
You just have to add an <img /> tag right after the first H1

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
        <h1> Title of your doc </h1>
        <img src="path/to/your/image.png">
    </body>
</html>

Paragraph

you want to add a simple paragraph to your document, just add a <p> tag right after a H1 tag.

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

        <title of the paragraph>
        <p>I'm a simple paragraph, I can contain <a href="#">link</a>
        and I'm formatted to make the reading experience easy</p>

    </body> 
</html>

Blockquote

Blockquotes are a good way to make an important text more visible, and with init.css the blockquotes are designed to be elegant and beautiful.

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

        <title of the paragraph>
        <p>I'm a simple paragraph, I can contain a bunch of text
        and I'm formated to make the reading experience easy</p>

        <blockquote>I'm a citation, I'm cool and beautiful</blockquote>

    </body> 
</html>

Links

To insert a simple link, just add a <a href="#"> inside a <p> tag, and it'll be recognized as a link in a text and not button

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

         <p>This paragraph contains a <a href="#">link</a> and 
an other <a href="#">link</a></p>

    </body> 
</html>

Buttons

If you want to place some buttons on your document, to link some resources or anything, you have to place a <a href="#"> but not inside a <p> or it'll be a simple link.

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

         <a href="#">button 1</a>
         <a href="#">button 2</a>
         <a href="#">button 3</a>

    </body> 
</html>

Images

You want to add images to your document, a simple <img src="#" /> will work.

But with init.css, if you add like 4 images, it'll be recognized as a small gallery, and it'll have an other look than if you add just one <img src="#"> tag.

<html>
    <head>
        <My super init.css website>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

         <a href="#">button 1</a>
         <a href="#">button 2</a>
         <a href="#">button 3</a>

    </body> 
</html>
