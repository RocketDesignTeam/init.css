init.css

The perfect responsive style for your project or reading pages

Introduction

With init.css, you only focus on inline tags. No need to add divs, or any bloc elements. If you want a kind of gihub pages site for a project and you only want to write the text.

The Title

Its really simple, the title of your document will be the first H1 of the page, and it'll stand on the header.

<html>
    <head>
        <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
        <h1> Title of your doc </h1>
    </body>
</html>

Header image

If you want to add a cool header image to give more power to your page, there is an easy way to do it.
You just have to add an <img /> tag right after the first H1

<html>
    <head>
         <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
        <h1> Title of your doc </h1>
        <img src="path/to/your/image.png">
    </body>
</html>
demo

Paragraph

you want to add a simple paragraph to your document, just add a <p> tag right after a H1 tag.

<html>
    <head>
         <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

        <h2>title of the paragraph</h2>
        <p>I'm a simple paragraph, I can contain <a href="#">link</a>
        and I'm formatted to make the reading experience easy</p>

    </body> 
</html>
demo

Blockquote

Blockquotes are a good way to make an important text more visible, and with init.css the blockquotes are designed to be elegant and beautiful.

<html>
    <head>
          <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

        <h2>title of the paragraph</h2>
        <p>I'm a simple paragraph, I can contain a bunch of text
        and I'm formated to make the reading experience easy</p>

        <blockquote>I'm a citation, I'm cool and beautiful</blockquote>

    </body> 
</html>

Links

To insert a simple link, just add a <a href="#"> inside a <p> tag, and it'll be recognized as a link in a text and not button

<html>
    <head>
         <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

         <p>This paragraph contains a <a href="#">link</a> and 
an other <a href="#">link</a></p>

    </body> 
</html>
demo

Buttons

If you want to place some buttons on your document, to link some resources or anything, you have to place a <a href="#"> but not inside a <p> or it'll be a simple link.

<html>
    <head>
          <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>

         <a href="#">button 1</a>
         <a href="#">button 2</a>
         <a href="#">button 3</a>

    </body> 
</html>

Images

Full width images

You want to add images to your document, a simple <img src="#" /> will work.

But there is a little logic about images. If you want to have a plain image on your article, a full width image, to illustrate your point, you can just place an <img src="#" /> tag right after a </p> or any header tag (<h1> -> <h6>) and you will have a full width image.

We recommend to use a rectangular image for a better effect.

 
<html>
    <head>
          <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
            
         <h1>A full width image with a h1 </h1>
         <img src="#" />

         <h2>A full width image with a h2 </h2>
         <img src="#" />
            
         <h3>A full width image with a h3 </h3>
         <img src="#" />

         <h4>A full width image with a h4 </h4>
         <img src="#" />
            
         <h5>A full width image with a h5 </h5>
         <img src="#" />

         <h6>A full width image with a h6 </h6>
         <img src="#" />
            
         <p>A full width image with a p </p>
         <img src="#" />
         <p>And you can go on </p>

     
    </body> 
</html>
demo

Pyramidal structure

If you want to show more than one image to illustrate your document, you just have to add more images, and the images following the first one, will have a width of 50%. And it'll create a kind of pyramidal structure. 

<html>
    <head>
          <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
            
         <h1>Pyramidal structure </h1>
         <img src="#" /> <!-- this is a 100% width img -->
         <img src="#" /> <img src="#" /> <!-- those are 50% width img -->
     
    </body> 
</html>
demo

Micro showcase

We wanted to use an inline tag to trigger the creation of a little gallery like structure.

For it you just have to place a <hr/> tag and right after you place your <img src="#" /> tags and it will have a special structure.

<html>
    <head>
          <title>My super init.css website</title>
        <link rel="stylesheet" type="text/css" href="init.css" />
    </head>

    <body>
            
         <h1>Micro showcase </h1>
        <hr/>
         <img src="#" /> <!-- this is a 50% width img -->
         <img src="#" /> <img src="#" /> <!-- those are 50% width img -->
         <img src="#" /> <img src="#" /> <!-- those are 50% width img -->
     
    </body> 
</html>
